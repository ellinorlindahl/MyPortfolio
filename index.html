How can I help you?

My goal in any design process is to create intentional, intuitive and invisible design by maintaining a solid understanding and connection with your users directly.

Although still soaking up everything I can in the different UX fields, I can feel my heart beat a little extra hard for information architecture (I love the pursuit of the ultimate logic, clarity and structure) and UX writing - where commucation and language meets design.

UX research: Connecting with your users and creating a deep understanding of their needs.
Information architecture: Organizing information in a structured and intuitive way.
UX writing: Designing a clear and helpful experience by using the right words.
Methods: Design Thinking / Qualitative interviewing / Quantintative surveys / Thematic analysis / Workshops / User Journeys / Impact mapping / User testing / and more.
Tools: Figma / Miro / Visual Studio Code / Balsamiq / and open to learning many more!
