Arbetsförmedlingen | 2020

A project focused on user testing.
Mission: As a part of 8 week course 'UX-process 2: User centered design', we did a group project for Arbetsförmedlingen.
The mission was to test and evaluate a specific existing user flow on Arbetsförmedlingens webpage.

      Methods used:
  • Design Thinking
  • In-depth user interviews
  • User testing
  • HMW workshop + Lightning Decision Jam
  • Impact Effort Mapping
